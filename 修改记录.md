### 3. 每轮 batch 进度条

用 tqdm 包裹 `train_loader`，实时显示：

- 当前 batch 进度（已处理 / 总数）
- 当前 loss 值
- 处理速度（it/s）
- 剩余时间估计

```
Epoch   1/200: |█████████████            | 45/120 [00:32<00:53, 2.34it/s] loss=0.5234
```

### 4. 每轮汇总行

合并原来的两行 print 为一行精华，核心指标一目了然：

```
★ Epoch   3/200 | loss:0.4123 | val auc:0.7821  val acc:0.7345 | best_auc:0.7821 (ep3) | 45.2s 2.7it/s | 早停计数: 0/10
  Epoch   4/200 | loss:0.4001 | val auc:0.7802  val acc:0.7321 | best_auc:0.7821 (ep3) | 44.8s 2.7it/s | 早停计数: 1/10
```

- `★` 标记当前最佳轮次
- 末尾 `早停计数` 显示距上次提升的轮数
